# Sistemas desarrollados

## Sistema Biblioteca

Permite:

- Registrar libros
- Registrar miembros
- Realizar préstamos
- Controlar disponibilidad de libros

---

## Sistema Facultad

Permite:

- Registrar estudiantes
- Aplicar herencia mediante Persona -> Estudiante
- Gestionar cursos y alumnos
- Validar datos ingresados

---

# Conceptos aplicados

- Clases y objetos
- Encapsulamiento
- Herencia
- Modularización
- Manejo de excepciones
- Validaciones
- Organización en módulos

---

# Manejo de excepciones

El sistema controla distintos errores mediante excepciones, por ejemplo:

- Nombre vacío
- Edad inválida
- Libro ya prestado
- Datos incorrectos
- Búsquedas inexistentes
